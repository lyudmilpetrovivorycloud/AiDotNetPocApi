As a developer, I want to build proof of Concepts (POCs) for key model types using the AiDotNet library, so that we can verify the library delivers on its claimed capabilities before adopting it for the project.

POCs are created for the following model types:
Named Entity Recognition (NER)
Classification
Synthetic generation
Each POC includes setup, configuration, and a working test scenario
Results from each POC are documented
Sample input/output is captured for each model type.
Findings are presented to the team for review.
